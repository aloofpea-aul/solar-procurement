📌 วิธี Deploy GAS:
1. เปิด Google Sheets → Extensions → Apps Script
2. วาง Code.gs → Save → รัน initializeSheets()
3. Deploy → New deployment → Web app → Execute as: Me → Who has access: Anyone
4. คัดลอก URL มาใส่ช่องด้านบน

💡 หมายเหตุ: ระบบเก็บข้อมูลใน Browser (localStorage) ก่อนเสมอ
