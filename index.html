เชื่อมต่อ Google Apps Script (GAS)

📌 วิธี Deploy GAS ครั้งแรก:
1. เปิด Google Sheets → Extensions → Apps Script
2. วาง Code.gs → Save (Ctrl+S)
3. เลือกฟังก์ชัน initializeSheets → กด ▶ Run → อนุมัติ permissions
4. Deploy → New deployment → Type: Web app
   Execute as: Me | Who has access: Anyone
5. คัดลอก URL มาใส่ช่องด้านบน → บันทึก

💡 ข้อมูลเก็บใน Browser ก่อนเสมอ — กด Sync ได้ทุกเมื่อ
