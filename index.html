วิธีตั้งค่า GAS Web App URL

วิธี deploy GAS:
1. เปิด Google Sheets ใหม่ → Extensions → Apps Script
2. วาง Code.gs และ appsscript.json
3. รัน initializeSheets() ครั้งแรก
4. Deploy → New deployment → Web app → Execute as: Me → Anyone
5. คัดลอก URL มาใส่ช่องด้านบน
